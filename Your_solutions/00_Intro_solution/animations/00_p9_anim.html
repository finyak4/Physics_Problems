Optimization Problem

A rectangle is constructed under the curve y = 3 - x² in the first quadrant. What are the dimensions of the rectangle with the maximum area?

Calculus Solution

1. Objective Function:

Area A = x · y

2. Constraint:

The point (x, y) is on the curve:
y = 3 - x²

3. Substitute y into Area:

A(x) = x(3 - x²) = 3x - x³

4. Find the derivative A'(x):

A'(x) = 3 - 3x²

5. Set A'(x) = 0 to find critical points:

3 - 3x² = 0
3x² = 3
x² = 1 → x = 1 (since x > 0)

6. Find maximum dimensions:

Width = 1
Height = 3 - (1)² = 2
Max Area = 1 · 2 = 2
