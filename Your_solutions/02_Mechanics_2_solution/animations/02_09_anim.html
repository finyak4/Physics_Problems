Analytical Solutions

1. Solving the Equation

We rewrite m(dv/dt) = -mg - kv by separating variables:

dv / (v + mg/k) = -(k/m) dt

Integrating from 0 to t (with v(0) = v₀):

v(t) = (v₀ + mg/k)e-(k/m)t - mg/k

Integrating velocity to find position y(t) with y(0) = y₀:

y(t) = y₀ + (m/k)(v₀ + mg/k)(1 - e-(k/m)t) - (mg/k)t

2. Maximum Height

Occurs when v(tpeak) = 0. Setting velocity to 0 and solving for t:

tpeak = (m/k) ln(1 + kv₀/mg)

Substituting tpeak into y(t):

ymax = y₀ + (m/k)v₀ - (m²g/k²) ln(1 + kv₀/mg)

3. Comparison (No Drag)

As k → 0, using Taylor series expansions, the formulas revert to standard kinematics:

ynd(t) = y₀ + v₀t - ½gt²
ymax,nd = y₀ + v₀² / 2g

Notice how the drag term explicitly reduces both the peak time and the maximum altitude.
