## 6. Function Analysis

Consider the function $f(x) = 3x^2 - 12x + 7$. Identify any local maxima or minima.

**Step-by-step solution:**

1. Find the first derivative to locate critical points:

   $f'(x) = 6x - 12$

2. Set the first derivative to zero and solve for $x$:

   $6x - 12 = 0 \implies 6x = 12 \implies x = 2$

3. Find the second derivative to determine concavity:

   $f''(x) = 6$
   Since $f''(2) = 6 > 0$, the function is concave up, meaning there is a local minimum at $x=2$.
   
4. Calculate the corresponding $y$-value:

   $f(2) = 3(2)^2 - 12(2) + 7 = 12 - 24 + 7 = -5$

**Answer:** Local minimum at $(2, -5)$.
