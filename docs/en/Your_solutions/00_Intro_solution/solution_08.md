## 8. Definite Integrals

Calculate the area under the curve of the function $f(x) = \sin(x)$ from $x=0$ to $x=\pi$.

**Step-by-step solution:**

1. Set up the definite integral:

   $\text{Area} = \int_{0}^{\pi} \sin(x) \, dx$

2. Find the antiderivative of $\sin(x)$, which is $-\cos(x)$.

3. Evaluate the definite integral using the Fundamental Theorem of Calculus:

   $\text{Area} = [-\cos(x)]_{0}^{\pi} = (-\cos(\pi)) - (-\cos(0))$

4. Substitute the known values ($\cos(\pi) = -1$, $\cos(0) = 1$):

   $\text{Area} = (-(-1)) - (-1) = 1 + 1 = 2$

**Answer:** The area is $2$.
