# 7. Standing Wave Modes

## Problem Statement
A standing wave with four antinodes is produced on a string of length $L = 80\text{ cm}$. What is the wavelength of this wave?

## Step-by-Step Solution

1. For a string fixed at both ends, a standing wave forms nodes at the boundaries. The distance between any two consecutive nodes (or consecutive antinodes) is exactly half of a wavelength ($\frac{\lambda}{2}$).

2. A standing wave with $n$ antinodes corresponds to the $n$-th harmonic.
   Therefore, the string's length $L$ fits exactly $n$ half-wavelengths:
   $$ L = n \frac{\lambda}{2} $$

3. In this problem, we are given:
   - Length $L = 80\text{ cm} = 0.8\text{ m}$
   - Number of antinodes $n = 4$

4. Substitute the values into the formula:
   $$ 0.8\text{ m} = 4 \times \frac{\lambda}{2} $$
   $$ 0.8\text{ m} = 2\lambda $$

5. Solve for the wavelength $\lambda$:
   $$ \lambda = \frac{0.8\text{ m}}{2} = 0.4\text{ m} $$

   The wavelength is 0.4 m (or 40 cm).
