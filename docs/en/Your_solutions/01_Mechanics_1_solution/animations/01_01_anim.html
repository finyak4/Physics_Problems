Mathematical Derivation

1. Differential Equations of Motion

Applying Newton's Second Law (F = ma) in horizontal (x) and vertical (y) directions:

ax = d²x / dt² = 0
ay = d²y / dt² = -g

Integrating with respect to time (t):

vx(t) = v₀·cos(θ)
vy(t) = v₀·sin(θ) - gt

Integrating again for position:

x(t) = (v₀·cos(θ))t
y(t) = (v₀·sin(θ))t - ½gt²

2. Time of Flight (T)

The projectile hits the ground when y = 0:

(v₀·sin(θ))T - ½gT² = 0
T = 2·v₀·sin(θ) / g
T = 12.27 s

3. Maximum Height (H)

Occurs when vertical velocity vy = 0:

tpeak = v₀·sin(θ) / g
H = y(tpeak) = (v₀·sin(θ))² / 2g
H = 184.62 m

4. Range (R)

Horizontal distance at t = T:

R = x(T) = v₀·cos(θ) · (2·v₀·sin(θ) / g)
R = v₀²·sin(2θ) / g
R = 979.84 m
