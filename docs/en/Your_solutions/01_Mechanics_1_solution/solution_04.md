## 4. Vector Calculus

Position $\vec{r}(t) = (3t^2)\hat{i} + (5t - 8t^2)\hat{j}$.

**Step-by-step solution:**
1. **Velocity vector:** Take the first derivative of position with respect to time.
   $\vec{v}(t) = \frac{d\vec{r}}{dt} = \frac{d}{dt}(3t^2)\hat{i} + \frac{d}{dt}(5t - 8t^2)\hat{j} = (6t)\hat{i} + (5 - 16t)\hat{j}$
2. **Acceleration vector:** Take the second derivative of position (or first of velocity).
   $\vec{a}(t) = \frac{d\vec{v}}{dt} = \frac{d}{dt}(6t)\hat{i} + \frac{d}{dt}(5 - 16t)\hat{j} = (6)\hat{i} + (-16)\hat{j}$

**Answer:** 
- Velocity: $\vec{v}(t) = (6t)\hat{i} + (5 - 16t)\hat{j}$
- Acceleration: $\vec{a}(t) = 6\hat{i} - 16\hat{j}$
